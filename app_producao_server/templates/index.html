Bem-vindo ao Projeto E-noé

Este projeto tem como objetivo monitorar rios urbanos utilizando tecnologias avançadas de sensoriamento remoto e inteligência artificial. Nosso foco é coletar dados multimodais, incluindo imagens e informações sobre o nível da água, para prever enchentes e contribuir para a mitigação de problemas ambientais em áreas urbanas.

Desenvolvido com apoio da FAPESP, este trabalho é parte de um esforço contínuo para integrar ciência e tecnologia em soluções práticas para problemas reais enfrentados pela sociedade.

Trabalhos Publicados

  • TDeep Learning and object detection for water level measurement using patterned visual markers
  • A deep learning workflow enhanced with optical flow fields for flood risk estimation
  • Water level identification with laser sensors, inertial units, and machine learning
